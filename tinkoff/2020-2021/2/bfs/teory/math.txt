1) 5x - 13y = 2 => 5x = 2 + 13y => x = (2+13y)/5 => 13y mod 5 = 3 составим табличку:
y mod 5   0 1 2 3 4
13y mod 5 0 3 1 4 2
значит y mod 5 = 1
y = 5n + 1
x = (2 + 65n + 13)/5 = 3 + 13n
2) 4x + 11y = 3 => 4x = 3 - 11y => x = (3 - 11y)/4 => 11y mod 4 = 3 составим табличку:
y mod 4   0 1 2 3
11y mod 4 0 3 2 1
значит y mod 4 = 1
y = 4n + 1
x = (3 - 44n - 11)/4 = (-8 - 44n)/4 = -2 - 11n
